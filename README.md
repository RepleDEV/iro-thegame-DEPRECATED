# iro-thegame
Iro game. Made using HTML, JS, and CSS. Quarantine project. Started in 04/15/2020. 

## Installation
Clone this repository, cd into it, and run npm i

```bash
git clone https://github.com/RepleDEV/iro-thegame
cd iro-thegame
npm i
``` 
## Usage

```bash
npm start
```
